pysmell.vim by webframp

Vundle compatible repo for PySmell

PySmell is an alternative omnicompletion library for Python. This is just a rebundling of the vim script to be compatible with Vundle/Pathogen usage. Main vim script page is here

Dependencies

PySmell

Install

Follow Pysmell setup instructions: http://code.google.com/p/pysmell/
